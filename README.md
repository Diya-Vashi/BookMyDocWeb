# BookMyDoc Web Platform

A professional healthcare SaaS landing page designed to showcase the BookMyDoc application.

## Overview
BookMyDoc is a modern, responsive website built to highlight the features, benefits, and user interface of the BookMyDoc medical application. It employs premium healthcare design aesthetics (a trusted blue & white color palette, clean typography, and smooth animations) to create a trustworthy experience for both clinics and patients.

## Features
- **Responsive Design**: Fully responsive layout built with Bootstrap 5, seamlessly adapting to mobile, tablet, and desktop screens.
- **Modern UI**: Clean, premium aesthetics featuring custom CSS variables, soft box-shadows, and modern rounded interfaces.
- **App Showcase**: An interactive carousel showcasing actual application screens to give users a preview of the native experience.
- **Smooth Animations**: Integrated scroll animations using AOS (Animate On Scroll) for a dynamic, polished feel.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 5.3.0
- AOS Animation Library
- FontAwesome Icons

## Getting Started
Since this is a static website, no complex build process or server configuration is required.

1. Clone or download this repository.
2. Open `index.html` in any modern web browser to view the site locally.

## Deployment
This project is deployment-ready for static hosting services such as GitHub Pages, Netlify, or Vercel. Simply push the files to your repository and enable static hosting pointing to the root directory.
